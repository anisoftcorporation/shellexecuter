Hello All

Welcome to Shell-Executor program. This script will enable you to execute any shell script in background from PHP. So you can trigger execution of any shell script from PHP front end and the shell script will be executed in the background. The current release is a very basic release and may not have many features.

Invite all to contribute and make it better.
